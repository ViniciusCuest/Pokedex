## Environment variables

To run this project, you're supposed to fill the following environment variables in your .env file

`API_KEY`

`ANOTHER_API_KEY`


## Running Locally

Project Clone

```bash
  git clone https://link-para-o-projeto
```

Directory of project

```bash
  cd pokedex
```

Install all dependencies

```bash
  npm install
```

Run the server

```bash
  npm run dev
```
